# Sales Order Management Workshop App

This is a dependency-free sales order management app for a one-day development workshop.

## How To Run

Open `index.html` in a browser.

## Current Epic Coverage

- Epic 1: Authentication and role access
- Epic 2: Dashboard
- Epic 3: Customer management
- Epic 4: Product management
- Epic 5: Sales order management
- Epic 6: Invoice and print view
- Epic 7: Search and filters
- Epic 8: Sample data
- Epic 9: Business UI foundation

## Workshop Notes

The app uses browser local storage for persistence. Use the `Reset Demo Data` button to restore sample customers, products, and orders.
